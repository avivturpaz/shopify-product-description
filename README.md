# Shopify Product Description Generator

> **Retired Venture experiment — historical reference only.**

A Flask prototype that generates Shopify product descriptions from a title, specifications, audience, and requested tone.

## What is implemented

- Groq-based text generation
- Multiple writing tones
- SQLite generation history
- Free-tier rate limiting
- Sentry and Venture alert hooks
- Polar checkout and Plausible integration points

## Status

Development stopped on July 10, 2026. The application is not maintained or production-ready. Generated copy and SEO claims require human review, and the external integrations depend on environment-specific credentials.

Created as part of the archived Venture product-experiment portfolio.
